## 3. 技术规格

| 属性 | 值 |
|------|-----|
| 输入模态 | **文本 + 图片 + 视频 + 音频 + PDF** |
| 最大输入 | **8,192 tokens**（跨所有模态合计）|
| 默认输出维度 | 3,072（已归一化）|
| 可选维度范围 | 128 ~ 3,072 (MRL) |
| 推荐维度 | 768, 1,536, 3,072 |
| 归一化 | 3,072-D 预归一化；其他维度需手动 L2 归一化 |
| 语言支持 | 100+ 语言 |
| Task Types | 8 种（与 001 相同）|
| Batch API | **暂不可用** |
| Vertex AI | **暂不可用** |

## 4. 多模态输入限制

| 模态 | 上限 | 支持格式 |
|------|------|---------|
| 文本 | 8,192 tokens | 纯文本 |
| 图片 | 6 张/请求 | PNG, JPEG |
| 音频 | 80 秒 | MP3, WAV |
| 视频 | 128 秒 | MP4, MOV (H264, H265, AV1, VP9) |
| PDF | 6 页 | PDF |

总约束：所有模态合计不超过 8,192 tokens。

## 5. 支持的 Task Types (8 种，与 001 相同)

| Task Type | 描述 |
|-----------|------|
| `SEMANTIC_SIMILARITY` | 文本相似度 |
| `CLASSIFICATION` | 文本分类 |
| `CLUSTERING` | 文本聚类 |
| `RETRIEVAL_DOCUMENT` | 文档索引 |
| `RETRIEVAL_QUERY` | 搜索查询 |
| `CODE_RETRIEVAL_QUERY` | 代码检索 |
| `QUESTION_ANSWERING` | QA 文档检索 |
| `FACT_VERIFICATION` | 事实核查 |

## 8. 新特性（相比 001）

### 8.1 多模态嵌入（最大突破）
将文本、图片、视频、音频、PDF 映射到**统一向量空间**，支持：
- 文本搜图片
- 图片搜文本
- 视频内容语义检索
- 跨模态相似度计算

### 8.2 输入上下文 4x 扩展
从 2,048 → 8,192 tokens，支持更长文档的单次嵌入。

### 8.3 跨模态聚合
单个 content 含混合 parts (text + image) → 生成一个聚合嵌入。
多个 contents → 各自独立嵌入。
Google 建议对复杂对象（如社交媒体帖子）使用平均法。

## 9. 嵌入空间兼容性

> **The embedding spaces between gemini-embedding-001 and gemini-embedding-2-preview are incompatible. This means you cannot directly compare embeddings generated by one model with embeddings generated by the other.**

— Google 官方文档

迁移必须**全量重新嵌入**所有数据，不存在渐进迁移路径。

## 10. SDK 用法 (Python)

```python
from google import genai
from google.genai import types

client = genai.Client(api_key="YOUR_API_KEY")

# 文本嵌入
result = client.models.embed_content(
    model='gemini-embedding-2-preview',
    contents='What is the meaning of life?',
    config=types.EmbedContentConfig(
        task_type='RETRIEVAL_DOCUMENT',
        output_dimensionality=768
    )
)

# 图片嵌入
result = client.models.embed_content(
    model='gemini-embedding-2-preview',
    contents=[types.Part.from_bytes(
        data=image_bytes, mime_type='image/png'
    )]
)

# 文本+图片聚合嵌入
result = client.models.embed_content(
    model='gemini-embedding-2-preview',
    contents=[types.Content(parts=[
        types.Part(text="An image of a dog"),
        types.Part.from_bytes(data=image_bytes, mime_type='image/png')
    ])]
)
```

## 13. 当前限制

1. **Preview 状态** — 可能有 breaking changes
2. **无 Vertex AI** — 无法通过 ADC 认证使用
3. **无 Batch API** — 大规模重嵌入需自行限流
4. **多模态 benchmark 未公布** — 跨模态检索精度未知
5. **嵌入空间不兼容** — 必须全量迁移
